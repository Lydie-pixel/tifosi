# Projet Base de Données - Tifosi

## Présentation

Ce projet consiste à créer et exploiter une base de données pour la gestion
d’une pizzeria nommée **Tifosi**.

Il comprend :
- La création du schéma relationnel
- Le peuplement de la base avec des données de test
- L’écriture de requêtes de vérification

---

## Contenu du projet

- schema.sql : création de la base, des tables et de l’utilisateur
- data.sql : insertion des données
- queries.sql : requêtes de test
- README.md : documentation

---

## Installation

1. Ouvrir MySQL
2. Exécuter le fichier `schema.sql`
3. Exécuter le fichier `data.sql`
4. Exécuter le fichier `queries.sql`

---

## Utilisateur MySQL

Un utilisateur est créé automatiquement :

- Nom : tifosi_user
- Mot de passe : Tifosi2026!

> Dans un contexte professionnel, les identifiants seraient stockés
> dans un fichier sécurisé (.env).

L’exécution du script peut nécessiter des droits administrateur.
